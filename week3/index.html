In Javascript, 'this' is used to refer to an object. For example, a function can use 'this' instead of specifying an object within the function. Using this method allows for code that's less easy to break.

An example of using 'this' in Javascript can be seen here:

The code used to accomplish this is:
let user = {
message: "This is an example message!",
showMessage() {
alert(this.message);
}
}
const clickMe = document.getElementById("this-example")
clickMe.addEventListener("click", () => {user.showMessage()})

There are several special object types like Math, Date, and RegExp.

In Javascript, you can manipulate an HTML document, or DOM. DOM is often represented by a bunch of nodes for each HTML element. This is a very common method of getting web pages to do things like having functional buttons or basic animations.

The example code from the 'this' demonstration shows an example of DOM manipulation. DOM manipulation can be used for more than just buttons, but that's one of the simpler examples.

Event listeners are how Javascript sees how a page is being interacted with. The demo code above contains an event listener that triggers a function when the element is clicked, but there are several other event listeners, such as on key presses or when the physical screen is interacted with.

Events can stop the default behavior of a key press or tap, but this isn't common or recommended practice.
